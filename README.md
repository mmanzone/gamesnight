# Games night
designed for families or friends wanting to track scores of popular card games from a simple web interface.

# Features
- Games are accessible via a modern browser from a desktop or computer
- Scores are stored temporarily in the browser cache to prevent loss in case of a page reload or exiting the app
- Players names are stored locally in the browser cache so they can be called back in the future. The list can be deleted by going to the settings on the home page
- All games have a french (default) and english user interface
- The results of all games can be shared as an image after the final round using the "share" features of the computer/phone/tablet used
- Games rules are available in the header menu for each game. The trashcan reset the game.

## Skull King
- the counting mechanism used is "rascal" without all the options
- Missed bets are displayed as empty circle; completed bets as filled circle and over-bets are showns as a + coin

## Papayoo
- Follows the official rules
- When the Papyoo is played, press the Papayoo button to record who got hit

## Dutch
(don't know the official name of this game)
- A successful "dutch" scores -10 for the person who called it
- A failed "dutch" scores the actual points + 10 points; if another player scores the same points as the player calling Dutch, the "dutch" is cancelled and standard points are counted

## Generic game
- This is used to track any kind of game cards
- The game ends whether a specific score or number of rounds is met
- The winner can be set to be the player with the most or the least amount of points at the end of the rounds/score match
